## Gyors publikálás

### GitHub Pages
1) Új publikus repo, pl. `portfolio`
2) A `portfolio/` mappa tartalma a repo gyökerébe
3) Settings → Pages → Deploy from branch (`main`), root
4) Link: `https://felhasznalonev.github.io/portfolio/`
